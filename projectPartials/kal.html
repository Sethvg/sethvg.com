Proposal

Description: kalE2E is a testing framework designed to provide end to end testing as well as test result reporting. It is built on top of the NodeJS framework, comes with custom logging levels and logging colors. It has recursive file searching to collect and compile the "steps (.js) and stories (.kalst)" files. It has environment specific preferences (ie: dev, test, uat, ect). Results will generate an HTML document locally, as well as send the results to a webserver where you can store them for auding purposes.
